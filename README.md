# Dev-assignment1

### 1-1 & 1-2 
과정을 진행하면서 처음에는 do-while문을 통해 "종료"란 명령어가 들어오기 전까지 반복하는 형태로
진행을 하고자 했으나 종료를 입력하게 되도 존재하지 않는 명령어란 내용이 나오게 되었다.

그에 따라 while(1)문으로 무한 반복 후 해당 루프 안에서 조건이 맞으면 break를 통해 바꾸는 식으로 
코드를 refactoring하게 되었다.
